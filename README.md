
# Installation:

## Clone:

    git clone git@github.com:Vuniverse0/fanWatch.git

## Run install.sh:

    sudo bash install.sh

**Requirements:**
 1. --   libnotify
 2. --  glib-2.0
 3. --  'gtk+-2.0'
 4. --  cmake 3.16+
 5. --  c++ 17 compiler
 
    ```
    sudo apt-get install libnotify-dev 
    sudo apt-get install libglib2.0-dev
    sudo apt-get install gtk+2.0
    sudo apt-get install cmake
